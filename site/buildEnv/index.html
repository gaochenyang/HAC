Cordova环境配置

  1. 首先安装nodeJS，请到官网https://nodejs.org/en/ 下载并安装
  2. 如果你是windows平台用户，还需要下载gitbash，后面自动化打包APP会用到
  3. 安装cordova
    npm install -g cordova
  4. 安装android sdk
  5. 安装adt
  6. 安装ant
